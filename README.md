# walkhttp
A CLI tool to call http endpoint with browser or prompt.

> [!NOTE]  
> Work in progress.. `walkhttp` is currently under development.

## Usage [Experimental]
```console
$ walkhttp --help
A CLI tool to call http endpoint with browser or prompt.

Usage:
  walkhttp [flags]

Flags:
      --help       Show help information
      --port int   port (default 3000)
      --version    Show version
```
