# walkhttp
A CLI tool to proxy HTTP request for rough observability.

## Usage
```console
$ walkhttp --origin https://example.com
┌─────────────────────────────────────────────────────────────────
│ walkhttp
│
│ Web console: http://localhost:3000/_
│ Origin URL: https://example.com
│
│ Try `curl http://localhost:3000/` and open web console.
└─────────────────────────────────────────────────────────────────
```
